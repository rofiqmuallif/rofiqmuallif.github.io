# Rofiq Muallif — Portfolio

A fast, accessible, and responsive portfolio built with semantic HTML, modern CSS, and vanilla JavaScript. No build process or framework is required, making it suitable for GitHub Pages.

## Project structure

- `index.html` — semantic page structure, SEO metadata, and structured data.
- `css/style.css` — compact visual design system and responsive layout.
- `css/accessibility.css` — keyboard, reduced-motion, and progressive-enhancement safeguards.
- `js/script.js` — optional, dependency-free interaction enhancements.
- `assets/` — add local production images and your CV here.

## Publish

Push the repository to GitHub, then enable **Settings → Pages → Deploy from a branch** and choose `main` with the `/ (root)` folder.

## Personalize before publishing

- Replace the two external placeholder images in `index.html` with optimized local images in `assets/`.
- Add a real CV at `assets/Rofiq-Muallif-CV.pdf`.
- Update the placeholder social and project URLs in `index.html`.
- Confirm the canonical URL and the JSON-LD `sameAs` links match your live profiles.
